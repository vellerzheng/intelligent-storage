##    智能存储

    2、该版本为重构后的微服务版本 ，详情请参见spring mvc旧版本. 
   https://github.com/vellerzheng/CloudSecuritybackup
## 功能
    
    实现文件切片，加密后上传到多个云服务器。并提供界面化文件管理系统和下载等功能。
    
## 架构
    
    论文提交后给出。

## 技术方案：

    spring boot & cloud 
    mybatis
    mysql
    redis
    rabbitmq
